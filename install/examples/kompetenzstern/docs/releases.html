11.9.20021.1.4
2.0a16-pre2
  • New user roles system
    • All features of the KS-Editor application -- especially the template builder feature -- can now be demonstrated without performing the "Steinbeis" customer installation. The "ks" and "admin" users are not required anymore. Note: KS-Editor 1.1.4 makes use of the user roles system as introduced in 2.0a15.
    • Every user can use the KS-Editor application by joining the "Kompetenzstern" workspace
    • Every user can be a "Template Builder" by setting the respective user role. If the role is set the default template ("Business Check") appears in the users personal workspace, ready for being edited. Further templates may be created. If the "Template Builder" role is removed the user can't no longer edit/create templates.
  • The Menus are polished in 3 regards
    • The new generic "Name" property (introduced in 2.0a15-pre8)
    • the "Show Side Panel" preference
    • the Build-Mode.
